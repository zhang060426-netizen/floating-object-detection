# MODULE_BOUNDARIES.md

## 1. 边界原则

本项目后续进入多 Agent 并行开发前，必须先明确模块边界。任何跨模块修改都要先说明影响范围，并同步共享契约文档。

核心原则：

1. **目录即所有权**：Agent 默认只修改自己负责目录。
2. **共享契约优先**：API、DB、模型输出、文件路径、JWT、指标格式变更必须先文档化。
3. **文档先行**：架构/API/数据库/AI 流程/模型结构重大变更必须先更新文档。
4. **小步可回滚**：不允许一次性大重构。
5. **缺失源码不补写**：当前阶段只标注待确认，不强行重建前后端源码。

## 2. 模块所有权矩阵

| Agent | 主目录 | 可负责内容 | 禁止事项 |
|---|---|---|---|
| Frontend Agent | `web-vue/` 或 `1项目代码/.../web-vue/` | Vue3 页面、Element Plus UI、ECharts、Pinia、路由、API 调用、检测结果展示、管理页面、UI 统一 | 不修改 Flask 业务逻辑、DB 字段、YOLO 推理/训练逻辑 |
| Backend Agent | `web-flask/` 或 `1项目代码/.../web-flask/` | Flask API、JWT、权限、文件上传、DB 访问、异步视频任务、实时接口、模型/数据集生命周期 | 不修改 Vue UI、不修改训练脚本、不直接替换模型权重 |
| AI Agent | `other/model_train/detect/` | YOLO 训练/验证/预测、模型评估、CLAHE、ByteTrack、推理性能、Qwen-VL 分析链建议 | 不修改业务页面、不改 DB/API 契约而不通知相关 Agent |
| Documentation Agent | 根目录文档、`3项目文档/`、`4测试包/` | README、架构、API、数据库、测试、部署、任务拆分、ER 图、流程图 | 不未经验证声明源码级事实，不覆盖业务代码 |

## 3. 共享契约

| 契约 | 影响模块 | 变更风险 |
|---|---|---|
| API 返回结构 | 前端、后端、测试 | 页面解析失败、兼容性破坏 |
| 数据库字段/表关系 | 后端、文档、测试 | 数据迁移、历史记录兼容问题 |
| `detection_result` JSON | 前端、后端、AI、报告 | 检测展示、报告、评估均受影响 |
| 模型类别定义 | 前端、AI、评估 | 类别名/颜色/统计错误 |
| 文件存储结构 | 前端、后端、文档 | 文件访问、报告导出、清理失败 |
| JWT 字段 | 前端、后端 | 登录状态、权限判断失效 |
| evaluation metrics 格式 | 前端、后端、AI、文档 | 评估展示和筛选错误 |
| Qwen-VL 分析输出 | 前端、后端、报告 | AI 分析展示与 Word 报告字段错位 |

## 4. 业务模块边界

### 用户与权限

后端负责登录、注册、JWT 签发/校验、角色权限、数据隔离；前端负责登录状态、路由守卫、菜单权限、用户信息展示；DB 负责 `user` 表字段定义和索引。

### 图片检测

前端负责上传、模型选择、结果图展示、目标列表、AI 分析展示、报告下载；后端负责文件保存、模型调用、裁剪图保存、检测记录入库、Qwen-VL 调用、报告生成；AI 负责 YOLO 输出格式、CLAHE 策略、分析 prompt 建议。

### 视频检测

前端负责视频上传、任务状态轮询、进度条、关键帧/结果视频展示；后端负责异步任务创建、帧采样、进度状态、关键帧保存、任务失败处理；AI 负责逐帧检测性能、采样策略、目标筛选策略。

### 实时检测

前端负责实时画面展示、会话启动/停止、检测统计展示；后端负责摄像头读取、会话管理、实时推理接口、记录保存；AI 负责模型推理速度、ByteTrack 跟踪、目标去重。

### 模型/数据集/评估

模型管理由后端和前端协作完成，AI Agent 提供训练产物规范和评估说明；数据集管理由后端负责验证，AI Agent 确认 YOLO 数据结构；模型评估由后端计算指标，前端展示，AI Agent 解释指标和阈值。

## 5. 跨模块变更流程

1. 发起 Agent 在对应文档中说明变更目的、影响模块和共享契约。
2. 受影响 Agent 阅读并确认接口/数据结构。
3. 实现前先更新测试用例或验证步骤。
4. 小步实现，保留回滚方案。
5. 提交前说明修改内容、影响范围、风险点、验证方式。

## 6. 待源码确认清单

- 前端实际页面目录、路由表、API 封装、全局状态。
- 后端实际 routes/services/models/dao 目录。
- 数据库初始化位置与真实字段是否与文档一致。
- 检测结果 JSON 真实结构。
- 文件上传与静态文件访问路径。
- Qwen-VL prompt、配置、错误处理。
- 视频任务是否使用子进程、线程、队列或数据库状态机。
