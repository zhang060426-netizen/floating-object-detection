# Evidence Levels

更新时间：2026-05-13  
阶段：Phase 2A：系统契约与重建基线  
边界：仅定义证据等级、标注规则和 Agent 输出格式；不修改业务代码、模型、数据库或测试资源。

## 1. 证据等级定义

| 证据等级 | 定义 | 可用来源 | 可声明范围 |
|---|---|---|---|
| 已源码确认 | 有实际业务源码证据，例如 `web-vue/src`、`web-flask/routes`、`web-flask/services`、DB 初始化脚本等 | 当前可见业务源码 | 可声明为当前实现事实 |
| 已资源确认 | 有测试包、数据集、权重、配置文件、依赖文件等资源证据，但不是业务源码 | `4测试包/`、`dataset/small_dataset/data.yaml`、`weights/*.pt`、`package.json`、`requirements.txt` | 可声明资源存在、配置存在、依赖意图存在 |
| 历史输出确认 | 有历史 output、指标文件、训练/验证图表等产物证据 | `other/model_train/detect/output/` | 可声明历史产物或历史指标存在 |
| 数据库文档确认 | 来自 `3项目文档/5数据库开发文档.md` 的表、字段、索引、SQL、初始化数据 | 数据库开发文档 | 可声明数据库文档定义，不可声明实际 DB 已创建 |
| 文档推断 | 来自系统介绍、架构、规划、说明文档的设计描述 | README、ROADMAP、ARCHITECTURE、系统介绍文档等 | 只能声明为设计预期或候选契约 |
| 待源码确认 | 当前缺少 `web-vue` / `web-flask` 业务源码，需要后续确认 | 缺失项、未找到路径、待恢复源码 | 不得声明为实现事实 |
| 冲突/差异 | 文档与当前可见文件存在明确不一致 | 文档描述与文件系统审计对比 | 可声明存在差异，并要求后续处理 |

## 2. 证据等级使用规则

| 使用场景 | 推荐证据等级 | 规则 | 证据等级 |
|---|---|---|---|
| 前端页面、路由、API 调用 | 待源码确认 | 当前缺少 `web-vue/src/`，不得声明页面实现已存在 | 待源码确认 |
| 后端 API、服务层、DB 初始化 | 待源码确认 | 当前缺少 Flask 业务源码，不得声明接口已实现 | 待源码确认 |
| 小数据集类别 | 已资源确认 | `data.yaml` 明确 `nc: 1`、`names: ['floating_object']` | 已资源确认 |
| 历史测试指标 | 历史输出确认 | 指标来自历史 output / 指标文件，不等于当前可运行验证结果 | 历史输出确认 |
| DB 表字段 | 数据库文档确认 | 只能说明数据库文档定义了这些表字段 | 数据库文档确认 |
| API 路径清单 | 文档推断 | 来自系统介绍 API 文档，需后端源码复核 | 文档推断 |
| 文档说有配置文件但文件系统未发现 | 冲突/差异 | 记录差异，不补写文件 | 冲突/差异 |

## 3. 禁止声明规则

| 禁止声明 | 原因 | 允许替代表述 | 证据等级 |
|---|---|---|---|
| “前端页面已经实现” | 当前未发现 `web-vue/src/views` | “文档描述存在该页面，待源码确认” | 待源码确认 |
| “后端接口已经可用” | 当前未发现 Flask routes | “文档描述存在该接口，待源码确认” | 待源码确认 |
| “数据库已按文档创建” | 当前未发现 DB 初始化源码或 SQLite 文件 | “数据库文档定义了该表结构” | 数据库文档确认 |
| “模型评估已通过” | Phase 2A 禁止执行测试 | “历史输出记录了该指标” | 历史输出确认 |
| “实时检测可运行” | 当前无后端入口且实时检测依赖本地 USB 摄像头 | “实时检测存在文档流程，需 Phase 2B 运行验证” | 文档推断 |
| “Qwen-VL 配置已存在” | 文档提到 `web-flask/algo/llm/config.py`，当前文件系统未发现 | “LLM 配置路径存在文档描述但源码缺失” | 冲突/差异 |

## 4. 字段标注模板

| 字段 | 类型 | 含义 | 来源 | 当前状态 | 证据等级 |
|---|---|---|---|---|---|
| `field_name` | `string` | 字段含义 | 输入文档或可见资源路径 | 必填/可选/待确认 | 待源码确认 |

字段说明必须同时包含：

- 字段名。
- 类型候选。
- 含义。
- 来源文档或资源路径。
- 当前状态。
- 证据等级。

## 5. 差异类型枚举

| 差异类型 | 定义 | 示例 | 证据等级 |
|---|---|---|---|
| 路径差异 | 文档逻辑路径与当前实际路径不一致 | `web-vue/` 实际位于 `1项目代码/floating-objects-detect-web/web-vue/` | 冲突/差异 |
| 源码缺失 | 文档描述了源码目录，但当前文件系统未发现 | 未发现 `web-flask/routes/` | 冲突/差异 |
| 接口/字段待核对 | 文档描述了接口或字段，但没有实现证据 | `/api/detection/detect` 待源码确认 | 待源码确认 |
| 依赖存在但实现待确认 | 依赖文件存在，业务实现缺失 | `requirements.txt` 有 `PyJWT`，但鉴权源码缺失 | 已资源确认 |
| 测试资源存在但自动化入口缺失 | 有资源，没有可执行测试脚本或可运行系统 | `4测试包/` 存在，但禁止执行冒烟 | 已资源确认 |
| 文档缺失 | 当前文档体系未覆盖该主题 | 独立 `FILE_STORAGE_CONTRACT.md` 尚未由 Backend 输出 | 待源码确认 |
| 历史产物缺口 | 历史输出引用文件缺失或只见占位文件 | 历史 `best.pt` 路径当前只有占位说明 | 冲突/差异 |

## 6. Agent 输出统一格式

| 章节 | 内容要求 | 证据等级 |
|---|---|---|
| 范围与输入 | 列出读取的文档、资源、目录，说明只读边界 | 文档推断 |
| 已确认事实 | 只写有源码、资源、数据库文档或历史输出支撑的事实 | 已资源确认 |
| 待源码确认 | 列出缺少业务源码导致不能确认的内容 | 待源码确认 |
| 差异与风险 | 记录文档与当前可见文件的不一致 | 冲突/差异 |
| 契约影响 | 标注影响 API、DB、模型输出、文件存储、JWT、metrics 的内容 | 文档推断 |
| Phase 2B 前置条件 | 写明进入代码重建前必须满足的门禁 | 文档推断 |
| 验证说明 | 只允许说明未执行测试及原因；不得伪造测试结果 | 待源码确认 |

统一输出声明：

```text
本输出为 Phase 2A 文档/契约交付；未修改业务代码、模型、数据库或测试资源；未执行端到端冒烟测试。
```
