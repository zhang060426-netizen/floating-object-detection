确认状态枚举：

- 已源码确认：有明确源码路径和实现证据。
- 已资源确认：有文件、目录、配置或测试资源证据，但不是业务源码。
- 文档已覆盖：文档存在并描述完整，但无源码证据。
- 待源码确认：当前缺少源码或入口文件。
- 文档缺失：当前文档体系没有覆盖。
- 冲突/差异：文档与源码或资源明确不一致。

差异类型枚举：

- 路径差异
- 源码缺失
- 接口/字段待核对
- 依赖存在但实现待确认
- 测试资源存在但自动化入口缺失
- 文档缺失
- 文档过期
