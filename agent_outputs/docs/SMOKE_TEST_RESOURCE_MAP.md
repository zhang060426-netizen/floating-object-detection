## 8. 冒烟测试前置条件

| 前置条件 | PASS 要求 | 当前状态 | 证据等级 |
|---|---|---|---|
| 前端可运行 | `npm run dev` 或等价启动成功 | 当前前端入口缺失 | 待源码确认 |
| 后端可运行 | Flask app 或等价服务启动成功 | 当前后端入口缺失 | 待源码确认 |
| API_CONTRACT 完成 | 图片/视频/评估/实时接口路径和响应格式明确 | 待输出 | 文档推断 |
| DB 初始化可用 | 最小表结构或开发 DB 策略明确 | 待输出 | 数据库文档确认 |
| 模型可加载 | 明确使用基础权重或补齐已训练权重 | 基础权重存在，已训练权重缺口 | 已资源确认 |
| 文件存储可用 | 原图、结果图、视频、报告路径策略明确 | 待契约输出 | 数据库文档确认 |
| 测试执行许可 | Phase 2B 明确允许后才能执行 | Phase 2A 禁止执行 | 文档推断 |
